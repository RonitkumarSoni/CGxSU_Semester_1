# 🛠 Basic CRUD Routes

| Method       | Endpoint                                   | Description                                   |
| :----------- | :----------------------------------------- | :-------------------------------------------- |
| **`GET`**    | `/api/v1/games`                            | Fetch all Steam games with pagination support |
| **`GET`**    | `/api/v1/games/:appid`                     | Fetch complete details of a specific game     |
| **`POST`**   | `/api/v1/games`                            | Create a new game entry                       |
| **`PUT`**    | `/api/v1/games/:appid`                     | Replace entire game record                    |
| **`PATCH`**  | `/api/v1/games/:appid`                     | Partially update game details                 |
| **`DELETE`** | `/api/v1/games/:appid`                     | Permanently delete a game                     |
| **`GET`**    | `/api/v1/games/:appid/exists`              | Check whether game exists                     |
| **`GET`**    | `/api/v1/games/:appid/summary`             | Get summarized details of a game              |
| **`GET`**    | `/api/v1/games/:appid/history`             | Retrieve update history of a game             |
| **`PATCH`**  | `/api/v1/games/:appid/archive`             | Archive a game entry                          |
| **`PATCH`**  | `/api/v1/games/:appid/restore`             | Restore archived game                         |
| **`GET`**    | `/api/v1/games/:appid/related`             | Fetch related game recommendations            |
| **`GET`**    | `/api/v1/games/:appid/screenshots`         | Fetch screenshots of a game                   |
| **`GET`**    | `/api/v1/games/:appid/trailers`            | Fetch trailers and gameplay videos            |
| **`GET`**    | `/api/v1/games/:appid/reviews`             | Fetch user reviews                            |
| **`POST`**   | `/api/v1/games/:appid/reviews`             | Add a user review                             |
| **`PATCH`**  | `/api/v1/games/:appid/reviews/:reviewId`   | Update user review                            |
| **`DELETE`** | `/api/v1/games/:appid/reviews/:reviewId`   | Delete user review                            |
| **`GET`**    | `/api/v1/games/:appid/system-requirements` | Fetch system requirements                     |
| **`GET`**    | `/api/v1/games/:appid/dlc`                 | Fetch downloadable content list               |
